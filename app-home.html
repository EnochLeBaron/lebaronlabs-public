LeBaronLabs Task Manager

A LeBaronLabs Tools app for project planning, task boards, shared workspaces, calendar views, files, messages, and connected workflow tools.

Purpose

LeBaronLabs Task Manager helps users organize work, collaborate in shared projects, manage boards and lists, view project calendars, and communicate through project messages.

Google Calendar

Users can connect their own Google Calendar account to display read-only calendar events inside the Calendar tab and calendar boxes.

The app does not create, edit, or delete Google Calendar events.

Launch

Task Manager is available through LeBaronLabs Tools for authorized users.
