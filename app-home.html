LeBaronLabs Task Manager

LeBaronLabs Task Manager is owned and operated by LeBaronLabs.

LeBaronLabs Task Manager helps users organize projects, task boards, shared workspaces, calendar views, files, messages, and connected workflow tools.

Users can import calendar feeds with iCal URLs and create subscription URLs so other calendar apps can display selected Task Manager calendar items.
